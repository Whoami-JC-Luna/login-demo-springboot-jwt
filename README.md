# login-demo-springboot-jwt

Demo project showcasing secure authentication with Spring Boot and JWT, following OWASP best practices.

## Tech stack

- **Backend** — Spring Boot 3, Spring Security, JWT
- **Frontend** — React + Tailwind CSS
- **Database** — PostgreSQL 17
- **Deploy** — Railway (backend) · Vercel (frontend)
